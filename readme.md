# Visual Acuity Tester using Optokinetic Nystagmus
Testing visual acuity by inducing Optokinetic Nystagmus
## Required modules
- pygame - generates screen
- svgwrite - creates .svg file of optotypes
- opencv-python -
- cairosvg - converts .svg to .png
